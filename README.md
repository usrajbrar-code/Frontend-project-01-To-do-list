# 📝 To-Do List App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript to manage daily tasks efficiently.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 📌 Real-time UI updates
* 💾 (Optional) Local storage support

---

## 🛠️ Tech Stack

* HTML → Structure
* CSS → Styling
* JavaScript → Functionality

---

## 📁 Project Structure

* `index.html` → Main structure of the app
* `style.css` → Styling and layout
* `script.js` → Logic and interactivity

---

## ⚙️ How It Works

* User enters a task in the input field
* Task is added to a list dynamically using JavaScript
* Each task has options to:

  * Mark as complete
  * Delete from the list

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start adding your tasks
